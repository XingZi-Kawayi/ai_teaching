# 极速上手AI应用
本项目包含AI应用开发相关的学习资料、算法实现和实战教程。

## 📁 目录结构
```
├── Basic_Algorithms/        # 基础算法实现
│   ├── basic_algorithms.md          # 算法知识汇总
│   ├── two_sum.py                   # 两数之和算法实现
│   └── length_of_longest_substring.py  # 最长无重复子串算法
└── Fine-tuning/             # 大模型微调相关
    └── LoRA/                       # LoRA微调技术
        └── 3小时LoRA微调最简路线图（含与RAG的区别）.md
```

## 📚 内容介绍

### 1. 基础算法
- 包含常见算法题的Python实现
- 配套算法思路讲解和复杂度分析
- 适合AI开发的编程基础训练

### 2. 大模型微调
- LoRA低秩适配技术完整教程
- 微调路线图和最佳实践
- LoRA与RAG技术对比分析
- 实际项目落地指导

## 🚀 快速开始
1. 克隆仓库到本地
2. 安装Python 3.8+环境
3. 进入对应目录查看具体内容

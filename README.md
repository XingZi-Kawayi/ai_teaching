## 📚 内容介绍

### 1. 基础算法
- 包含常见算法题的Python实现
- 配套算法思路讲解和复杂度分析
- 适合AI开发的编程基础训练

### 2. 大模型微调
- LoRA低秩适配技术完整教程
- 微调路线图和最佳实践
- LoRA与RAG技术对比分析
- 实际项目落地指导

### 3. LangChain框架
- LangChain核心组件系统学习
- 提示词模板、输出解析器实战
- Chain链和Memory记忆模块讲解
- 字节跳动LLM调用实战示例
- 从入门到精通的完整学习路径

### 4. RAG检索增强生成
- RAG技术原理和核心组件
- 文档加载、文本切割、向量嵌入全流程
- 向量数据库使用和检索优化
- 对话链和文档链实战开发
- 企业级知识库搭建指导

### 5. Agent智能体
- 智能体核心概念和工作原理
- 工具调用、规划推理实战
- 多智能体协作系统开发
- 实际业务场景落地案例
*目录内容持续更新中*
